# Change Log

All notable changes to the "Interlaken Pro" theme extension are documented in this file.

## [1.1.0] - 2024-11-20

### Changed
- Updated all repository URLs to point to new `interlaken-pro` repository
- Updated GitHub links in README and package.json

## [1.0.9] - 2024-11-20

### Fixed
- Cleaned up themes folder to only include Interlaken Pro variants

## [1.0.8] - 2024-11-20

### Added
- Preview screenshots for both dark and light themes
- Updated README with screenshot showcase

## [1.0.7] - 2024-11-20

### Added
- Professional Interlaken Pro logo
- Renamed all theme files to match new branding

### Changed
- Updated package.json with new theme names and descriptions
- Enhanced README with logo and improved documentation

## [1.0.6] - 2024-11-20

### Added
- Consistent F0F1F2 background for light theme editor and terminal
- Warm background color for better visual consistency

### Changed
- Light theme background updated for warmth and eye comfort

## [1.0.2] - 2024-11-20

### Fixed
- JavaScript/TypeScript operator colors (instanceof, new) now match function colors
- Boolean literals (true/false) properly colored
- Go declaration keywords (type, struct, map, package, func, nil) now match interface colors

## [1.0.1] - 2024-11-20

### Added
- Interlaken Pro Light theme with Default Light Modern pattern
- Comprehensive syntax highlighting for multiple languages
- Consistent color hierarchy across all supported languages
- Professional dark theme with excellent syntax highlighting
- Support for JavaScript, TypeScript, Go, Python, C++, and more languages

### Fixed
- String colors corrected to orange (#C41E3A) across all languages
- Go import colors differentiated from type names
- Template literal syntax highlighting improvements
